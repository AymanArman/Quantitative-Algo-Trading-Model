User is new to algorithmic trading — no prior experience building a trading model. Strong conceptual thinker who reasons through problems carefully before committing to decisions.

Understands statistical concepts well when built up from first principles with simple language. Does not need hand-holding but benefits from concrete examples (e.g. actual column structures, worked numeric examples).

Reads Ernest Chan's *Algorithmic Trading: Winning Strategies and Their Rationale* (2013) — familiar with his momentum and lookback methodology. Reference this when relevant.

**How to apply:** Don't assume trading domain knowledge. Explain mechanics concretely. When introducing a new concept, ground it in what the user already understands before building on it.
